# RPG — domain

Fatia "XP e nível" de RPG_SYSTEM.md §1 (participação/jornada — não
libera exercício, isso é papel do domínio de habilidade em
`features/progression`). `xp_rules.dart` decide créditos de XP a partir
de fatos de uma sessão concluída; `level_curve.dart` traduz XP total em
nível/progresso. Sem UI/rede/banco.

Referência: `06_GAMIFICATION/RPG_SYSTEM.md` §1-3,
`06_GAMIFICATION/ECONOMY_AND_ANTI_ABUSE.md` §2-3.
